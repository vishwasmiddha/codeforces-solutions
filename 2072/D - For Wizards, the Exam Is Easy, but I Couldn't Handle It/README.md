Help him enroll and solve the problem!</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$An inversion in an array $$$b$$$ of length $$$m$$$ is defined as a pair of indices $$$(i, j)$$$ such that $$$1 \le i  \lt  j \le m$$$ and $$$b_i  \gt  b_j$$$.
