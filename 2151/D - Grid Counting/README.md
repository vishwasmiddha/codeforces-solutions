Grid Counting</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><div class="epigraph"><div class="epigraph-text"><span class="tex-font-style-it"><a href="https://soundcloud.com/thorn-557109993/paragonx9-hyperioxx-id-223469">ParagonX9 - HyperioxX</a></span></div><div class="epigraph-source">⠀</div></div><p>Alice has a grid of size $$$n \times n$$$. Initially, all cells are colored white. Alice wants to color some cells black satisfying certain properties.</p><p>Let $$$(x_1, y_1), (x_2, y_2), \ldots, (x_m, y_m)$$$ be the cells colored black. You are given an array $$$a$$$ of size $$$n$$$. The following conditions must be satisfied:</p><ul> <li> For each row $$$k$$$ ($$$1 \le k \le n$$$), there exists exactly $$$a_k$$$ indices $$$i$$$ ($$$1 \le i \le m$$$) such that $$$x_i = k$$$. Less formally, there should be $$$a_k$$$ black cells in row $$$k$$$. </li><li> For each $$$k$$$ ($$$1 \le k \le n$$$), there exists exactly one index $$$i$$$ such that $$$\max(x_i, y_i) = k$$$. </li><li> For each $$$k$$$ ($$$1 \le k \le n$$$), there exists exactly one index $$$i$$$ such that $$$\max(x_i, n + 1 - y_i) = k$$$. </li></ul><p>For example, when $$$a = [2, 2, 1, 0, 0]$$$, a possible set of black cells is $$$\{(1, 1), (2, 2), (3, 3), (2, 4), (1, 5)\}$$$. This corresponds to the following grid:</p><center> <img class="tex-graphics" src="https://espresso.codeforces.com/15302df2da444ae01b9f8661ba24058c32c72e12.png" style="max-width: 100.0%;max-height: 100.0%;" width="265px">   <span class="tex-font-size-small">5×5 grid with specified black cells.</span> </center><p>Count the number of possible grids. $$$2$$$ grids are said to be different if there is some cell which is colored black in one of the grids, and white in the other. Since the answer may be large, output it modulo $$$998\,244\,353$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. $$$n$$$ integers — $$$a_1, a_2, \ldots, a_n$$$ ($$$0 \le a_i \le n$$$)</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div
