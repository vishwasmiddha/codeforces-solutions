There is a string $$$s$$$ of length $$$n - 1$$$ that describes the requirements of the arrangement.</p><p>Specifically, for each $$$i$$$ from $$$1$$$ to $$$n - 1$$$:</p><ul><li> If $$$s_i = \texttt{ \lt }$$$, then the length of the stick at position $$$i + 1$$$ must be <span class="tex-font-style-bf">smaller</span> than all sticks before it; </li><li> If $$$s_i = \texttt{ \gt }$$$, then the length of the stick at position $$$i + 1$$$ must be <span class="tex-font-style-bf">larger</span> than all sticks before it.</li></ul><p>Find any valid arrangement of sticks. We can show that an answer always exists.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases.
