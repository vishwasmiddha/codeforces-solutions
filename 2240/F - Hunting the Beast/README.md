## Problem Statement

<div class="header"><div class="title">F. Hunting the Beast</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p><span class="tex-font-style-it">In Zhuji, a city in central Zhejiang Province, local folklore tells of a wild beast known as the <span class="tex-font-style-underline">Modeiyon</span>. Dwelling deep in the mountains, it is said to sneak into villages at night to devour livestock and prey on lone travelers. Though many elders claim to have seen it, no photograph of the creature has ever been taken.</span></p><p>A brave group of $$$m$$$ people decides to head up the mountain to hunt the beast. The mountain's locations and trails can be modeled as a <span class="tex-font-style-bf">functional graph</span> $$$G$$$ with $$$n$$$ vertices (numbered $$$1$$$ to $$$n$$$). A functional graph is a directed graph with $$$n$$$ vertices and $$$n$$$ edges, where every vertex has an out-degree of exactly $$$1$$$. Additionally, it is known that the mountain's trails <span class="tex-font-style-bf">do not ever form self-loops</span>.</p><p>The group will choose exactly $$$m$$$ distinct vertices to form their initial starting set $$$S$$$. A starting set $$$S$$$ is defined as <span class="tex-font-style-it">successful</span> if every vertex $$$u$$$ in the graph is reachable from at least one vertex $$$v \in S$$$ (a vertex is always reachable from itself). </p><p>There are $$$\binom{n}{m}$$$ possible ways to choose the starting set of size $$$m$$$. They define the <span class="tex-font-style-it">value</span> of a graph $$$G$$$ as the number of <span class="tex-font-style-it">successful</span> starting sets it has.</p><p>However, the exact layout of the mountain's trails is unknown. If the destination of the single outgoing edge from each vertex is chosen arbitrarily from the remaining $$$n-1$$$ vertices (excluding the vertex itself), there are exactly $$$(n-1)^n$$$ possible functional graphs. Given $$$n$$$ and $$$m$$$, your task is to calculate the sum of the values of all $$$(n-1)^n$$$ possible functional graphs. Since the answer can be very large, print it modulo $$$998\,244\,353$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows.</p><p>The only line of each test case contains two integers $$$n,m$$$ ($$$1\le m \le n\le 10^6$$$) — the number of vertices in the graph and the number of starting vertices.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$10^6$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer — the sum of the values of all $$$(n-1)^n$$$ possible functional graphs, modulo $$$998\,244\,353$$$.</p></div><div
