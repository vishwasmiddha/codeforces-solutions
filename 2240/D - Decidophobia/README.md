## Problem Statement

<div class="header"><div class="title">D. Decidophobia</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>There are $$$n$$$ people attending a round-table party, numbered $$$1, 2, 3, \ldots, n$$$ in clockwise order. You have prepared some gifts to distribute among them. </p><p>Each person $$$i$$$ has a weight $$$a_i$$$ and a common field of view $$$d$$$. The field of view for person $$$i$$$ consists of the $$$d$$$ people sitting clockwise and the $$$d$$$ people sitting counter-clockwise from them (a total of $$$2d$$$ people excluding person $$$i$$$). </p><p>The happiness gained by person $$$i$$$ is determined by the following rules: </p><ul> <li> If person $$$i$$$ receives a gift, and there are $$$x$$$ people within their field of view who did not receive a gift, they gain $$$x \cdot a_i$$$ happiness. </li><li> If person $$$i$$$ does not receive a gift, and there are $$$x$$$ people within their field of view who received a gift, they incur $$$-x \cdot a_i$$$ happiness. </li></ul><p>You want to maximize the total happiness of all $$$n$$$ people combined. Find this maximum value.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows.</p><p>The first line of each test case contains two integers $$$n$$$ and $$$d$$$ ($$$3 \le n \le 2 \cdot 10^5$$$, $$$1 \le d  \lt  \frac{n}{2}$$$). $$$n$$$ over all test cases does not exceed $$$10^6$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer representing
