The difference between the versions is that in this version, the constraints on $$$n$$$, $$$x$$$, $$$s$$$, $$$t$$$ are smaller. You can hack only if you solved all versions of this problem.</span> </p><p>Alice's friends have come to the party, and now they are lining up to enter the party.</p><p>There are $$$x$$$ tables at the party with $$$s$$$ seats each. Each seat can only hold one person.</p><p>Each friend has one of these three following personalities: </p><ul> <li> Introverts (<span class="tex-font-style-tt">I</span>) who have to sit at an empty table </li><li> Extroverts (<span class="tex-font-style-tt">E</span>) who have to sit at a non-empty table </li><li> Ambiverts (<span class="tex-font-style-tt">A</span>) who can sit at any table. </li></ul><p>Initially, every seat is empty. However, because Alice was eating cakes, her friends had already formed a line, which Alice cannot change their order. For each person in the line, Alice has to assign them a table or kick them out of the party. Each person is seated before the next person is assigned a table.</p><p>Wanting to have a lot of fun at the party, Alice needs to seat as many people as she can at the party. Help her find the maximum number of friends she can have at the party. </p><p>Note that once a friend is seated, they are not allowed to move even if they are not seated according to their personality anymore. </p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The second line contains a string $$$u$$$ of length $$$n$$$ consisting only of the letters <span class="tex-font-style-tt">A</span>, <span class="tex-font-style-tt">E</span>, and <span class="tex-font-style-tt">I</span>, representing an ambivert, extrovert, and introvert respectively.</p><p>It is guaranteed that the sum of $$$n$$$ for all test cases is at most $$$3000$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output an integer: the maximum number of people seated.</p></div><div
