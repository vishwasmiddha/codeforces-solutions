For example, the strings <span class="tex-font-style-tt">"codeforces"</span>, <span class="tex-font-style-tt">"string"</span> are <span class="tex-font-style-it">anti-palindromes</span>, but the strings <span class="tex-font-style-tt">"abacaba"</span>, <span class="tex-font-style-tt">"abc"</span>, <span class="tex-font-style-tt">"test"</span> are not.</p><p>Determine the minimum number of operations required to make the string $$$s$$$ an <span class="tex-font-style-it">anti-palindrome</span>, or output $$$-1$$$, if this is not possible.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.
