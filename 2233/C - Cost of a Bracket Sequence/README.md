## Problem Statement

<div class="header"><div class="title">C. Cost of a Bracket Sequence</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p>Let the cost of an arbitrary bracket string be the length of it's longest subsequence$$$^{\text{∗}}$$$ that is a regular bracket sequence$$$^{\text{†}}$$$.</p><p>You are given a bracket string $$$s$$$ and an integer $$$k$$$. Your task is to remove at most $$$k$$$ characters from the string $$$s$$$ so that the cost of the resulting string is minimized.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A sequence $$$a$$$ is a subsequence of a sequence $$$b$$$ if $$$a$$$ can be obtained from $$$b$$$ by the deletion of several (possibly, zero or all) element from arbitrary positions.</p><p>$$$^{\text{†}}$$$A bracket sequence is called regular if it is possible to obtain a correct arithmetic expression by inserting the characters $$$+$$$ and $$$1$$$ into this sequence. For example, the sequences "$$$\texttt{(())()}$$$", "$$$\texttt{()}$$$", and "$$$\texttt{(()(()))}$$$" are regular, while "$$$\texttt{)(}$$$", "$$$\texttt{(()}$$$", and "$$$\texttt{(()))(}$$$" — are not.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^3$$$). The description of the test cases follows.</p><p>The first line of each test case contains two integers $$$n$$$ and $$$k$$$ ($$$1 \le n \le 5\,000$$$; $$$0 \le k \le n$$$) — the length of the string $$$s$$$ and the maximum number of deletions.</p><p>The second line of each test case contains a string $$$s$$$ of length $$$n$$$ consisting of the characters "$$$\texttt{(}$$$" and/or "$$$\texttt{)}$$$".</p><p>Additional input constraints: </p><ul> <li> the sum of $$$n$$$ over all test cases does not exceed $$$5\,000$$$. </li></ul></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a binary string of length $$$n$$$. The $$$i$$$-th character should be equal to "<span class="tex-font-style-tt">1</span>" if the corresponding character of the string $$$s$$$ is removed, and "<span class="tex-font-style-tt">0</span>" otherwise.</p><p>The number of ones in the string must not exceed $$$k$$$. The cost of the string obtained after removing the marked characters must be as small as possible.</p><p>If there are several answers, output any of them.</p></div><div class="sample-tests"><div
