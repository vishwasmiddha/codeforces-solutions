<div class="header"><div class="title">D. They have $$$(n+1)$$$ boxes of decorations, numbered from $$$0$$$ to $$$n$$$. Initially, the $$$i$$$-th box contains $$$a_i$$$ decorations.</p><p>We say that a permutation $$$p$$$ of size $$$n$$$ (an array of size $$$n$$$ where each number from $$$1$$$ to $$$n$$$ appears exactly once) is <span class="tex-font-style-bf">fair</span> if it is possible to hang all the decorations on the tree using the following process: </p><ul> <li> person $$$p_1$$$ takes a decoration either from box $$$0$$$ or from box $$$p_1$$$, and hangs it on the tree; </li><li> person $$$p_2$$$ takes a decoration either from box $$$0$$$ or from box $$$p_2$$$, and hangs it on the tree; </li><li> and so on; </li><li> after person $$$p_n$$$, person $$$p_1$$$ follows, and the process repeats until all the decoration are on the tree. </li></ul><p>During this process, there should never be a situation where person $$$i$$$ needs to hang a decoration, but both box $$$0$$$ and box $$$i$$$ are empty. If such a situation cannot be avoided, the permutation is not fair. However, if the people can choose from which box to take a decoration during each step in such a way that this situation does not arise, then the permutation is fair.</p><p>Your task is to calculate the number of fair permutations. Since the answer may be large, output it modulo $$$998244353$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer $$$t$$$
