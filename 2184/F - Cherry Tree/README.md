The vertices of the tree are numbered with integers from $$$1$$$ to $$$n$$$. The root of the tree is vertex number $$$1$$$.</p><p>In each leaf$$$^{\text{†}}$$$ of the tree, there grows one cherry. You want to collect all the cherries, and to do this, you perform the following action several times:</p><p>You choose any vertex of the tree $$$v$$$ (including the root or a leaf) and "shake" it. After that, cherries fall from all the leaves that are descendants$$$^{\text{‡}}$$$ of vertex $$$v$$$ (if vertex $$$v$$$ itself is a leaf, then a cherry falls from it). If cherries have already fallen from any leaf before, the tree will break, so such a situation must be avoided.</p><p>According to an ancient legend of the cherry orchard, the number of vertices you shake should be a multiple of three.</p><p>Is it possible to collect all the cherries in this way?</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A tree with $$$n$$$ vertices is an undirected connected graph with $$$n$$$ vertices and $$$n - 1$$$ edges. A rooted tree is a tree in which one of the vertices is special and is called the root.</p><p>$$$^{\text{†}}$$$A leaf is a vertex that has no descendants.</p><p>$$$^{\text{‡}}$$$The descendants of vertex $$$v$$$ are all vertices $$$u \neq v$$$ such that on the shortest path from the root to $$$u$$$, vertex $$$v$$$ is encountered.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of several test cases. The first line contains a single integer $$$t$$$ $$$(1 \le t \le 10^4)$$$ — the number of test cases. The following lines describe the test cases.</p><p>The first line of each test case contains a single integer $$$n$$$ ($$$2 \leq n \leq 2 \cdot 10^5$$$).</p><p>The next $$$n - 1$$$ lines of each test case contain two integers $$$u$$$ and $$$v$$$ ($$$1 \leq u, v \leq n, u \neq v$$$) — the vertices connected by the next edge of the tree.</p><p>It is guaranteed that the graph in each data set is a tree.</p><p>It is guaranteed that the sum of $$$n$$$ across all input data sets does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each input data set, output "<span class="tex-font-style-tt">YES</span>" on a separate line if it is possible to collect all the cherries. Otherwise, output "<span class="tex-font-style-tt">NO</span>".</p><p>You can print each letter in any case (upper or lower). For example, "<span class="tex-font-style-tt">YeS</span>", "<span class="tex-font-style-tt">no</span>" and "<span class="tex-font-style-tt">yES</span>" are all acceptable.</p></div><div
